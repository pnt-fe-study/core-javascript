1.  일정 : <u>**2025년 4월 10일 ~ 2025년 5월 22일까지**</u> 정리 완료를 목표로 해요.
2.  작성 방법 : 각 해당 사람의 폴더 내 챕터 별 디렉토리에 **마크다운 파일**로 학습한 내용을 정리해요.
3.  마감 기한 : 스터디 시작 전인 <u>**목요일 오전 11시 전(점심 스터디 시작 전)까지**</u> 작성을 완료해 올려요.
